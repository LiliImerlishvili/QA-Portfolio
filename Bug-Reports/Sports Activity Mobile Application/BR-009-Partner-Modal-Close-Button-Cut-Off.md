# BR-009 — Close Button Is Cut Off in Partner Details Modal

## Summary

The close button appears clipped and misaligned in the partner details modal.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iPhone 14 Pro |
| Operating System | iOS 26.4.2 |
| Application Version | 1.0 (build 7) |
| Reproducibility | 100% |

## Severity

Medium

## Priority

Normal

## Preconditions

- The user is authenticated unless otherwise stated.
- The relevant feature is available.

## Steps to Reproduce

1. Open Marketplace.
2. Open Partners.
3. Select a partner.
4. Observe the close button.

## Actual Result

The close button is partially cut off.

## Expected Result

The close button should be fully visible and correctly aligned.

## User Impact

The modal appears broken and may be difficult to close.

## Attachment

Evidence available in the original defect management system.
