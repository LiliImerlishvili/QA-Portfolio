**Goal:**  
Validate that a new user can register, log in, and add a product to Favorites.

**Preconditions:**  
- Email not used in the system.  

**Flow:**  
1. User opens Registration page.  
2. User fills valid data and creates an account.  
3. User logs into the account (if not auto-logged in).  
4. User opens product listing and selects a product.  
5. User adds product to Favorites.  
6. User opens Favorites page and checks that product is listed.
