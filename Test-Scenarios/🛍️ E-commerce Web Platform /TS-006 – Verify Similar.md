# TS-006 – Verify Similar and Recommended Products Relevance

**Goal:**  
Check that Similar/Recommended products match the current product’s brand and category.

**Preconditions:**  
- Recommendation logic is configured.  

**Flow:**  
1. User opens a specific product (e.g. branded shoes).  
2. User scrolls to Similar Products / Recommended section.  
3. User reviews items displayed.  
4. User checks that items match current product’s brand or category.
