**ID:** BUG-002

## Description
Uploaded images are automatically cropped, cutting off important parts of the image.

## Preconditions
- Invitation editor open
- Image upload available

## Steps to Reproduce
1. Upload an image with custom dimensions
2. Place image on invitation
3. Open preview

## Actual Result
Image appears cropped incorrectly.

## Expected Result
Image should retain original proportions or allow manual cropping.
