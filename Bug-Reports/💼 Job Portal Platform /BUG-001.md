**Bug ID:** BUG-001  
**Priority:** High  
**Severity:** Critical  
**Status:** Open  

## Description
The registration form allows users to submit the form without filling in mandatory fields, resulting in invalid user accounts being created.

## Preconditions
- User is not logged in
- Registration page is accessible

## Steps to Reproduce
1. Open the Registration page
2. Leave Email field empty
3. Leave Password field empty
4. Click the "Register" button

## Actual Result
The form is submitted successfully and a user account is created.

## Expected Result
The form submission should be blocked and validation messages should be displayed for all mandatory fields.

## Environment
- Browser: Chrome 121
- OS: macOS
- Environment: Staging
