# TC-RE-007 - Cross-Browser UI Consistency

**Priority:** Medium

## Preconditions
- Test environments are available in Chrome, Firefox, Safari, and Edge
- Same property page can be opened

## Steps
1. Open the same property details page in each browser
2. Compare layout, fonts, spacing, and CTA placement
3. Interact with gallery and inquiry form in each browser

## Expected Result
- UI layout is consistent across browsers
- No broken components or misaligned buttons appear
- Core interactions work in all browsers
