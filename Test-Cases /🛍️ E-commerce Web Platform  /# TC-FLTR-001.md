# TC-FLTR-001 – Category Filter Shows Only Matching Products

**Purpose:**  
To verify that applying a category filter returns only relevant products.

**Related Bug:** UI-FLTR-093  
**Module / Feature:** Filters  
**Priority:** High  

---

### Steps:
1. Open product listing.  
2. Apply a specific category filter (e.g., "Shoes").  
3. Scroll through products.  

---

### Expected Result:
- All products in the list belong to the selected category.
