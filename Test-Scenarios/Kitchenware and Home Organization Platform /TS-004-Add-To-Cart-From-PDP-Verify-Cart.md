# TS-004 - Add to Cart from Product Page and Verify Cart

## Goal
Verify that user can add a product to cart from PDP and cart reflects correct item details.

## Preconditions
- Website is available.
- Product is in stock.
- Cart is enabled.

## Flow (High-level)
1. User opens a product detail page.
2. User selects required variants (if applicable).
3. User clicks "Add to Cart".
4. User opens cart (drawer or cart page).
5. User verifies product name, quantity, price, and totals are correct.
