# BR — Deleted Words Reappear During Typing

## Summary

Words removed with Backspace reappear unexpectedly while the typing exercise continues.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iPhone 14 Pro |
| Operating System | iOS 26.4.1 |
| Application Version | 2.0.0 (1.1.12) |
| Reproducibility | 100% |

## Severity

Critical

## Priority

Urgent

## Preconditions

- A typing exercise is active.

## Steps to Reproduce

1. Start typing a practice text.
2. Type approximately 20 characters or words.
3. Delete part of the entered content using Backspace.
4. Continue typing.

## Actual Result

Previously deleted content reappears or is restored unexpectedly.

## Expected Result

Deleted content should remain deleted unless the user types it again.

## User Impact

The exercise state becomes unreliable and typing results may be incorrect.

Evidence available in the original defect management system.
