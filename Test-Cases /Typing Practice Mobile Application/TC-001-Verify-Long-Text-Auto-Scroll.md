# TC-001 — Verify Long Text Auto Scroll

## Related Defect

- BR-001

## Objective

Verify that long typing exercises keep the active line visible.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iPhone 14 Pro |
| Operating System | iOS 26.4.1 |
| Application Version | 2.0.0 (1.1.12) |
| Reproducibility | 100% |

## Test Steps

1. Open a long practice text.
2. Start typing.
3. Continue for multiple lines.
4. Observe viewport and active line position.

## Expected Result

- The active line remains visible.
- The screen auto-scrolls smoothly.
- No manual scrolling is required.

## Status

Not Executed
