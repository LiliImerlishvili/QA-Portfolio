# Vacancy Review and Community Feedback Platform — Bug Reports

This folder contains NDA-safe bug reports based on real defects identified during manual QA testing of a vacancy review and community feedback platform.

All sensitive information has been removed or generalized, including the original platform name, URLs, vacancy and company names, user identities, comments, internal task statuses, employee details, identifiers, and proprietary implementation information.

All defects documented here were identified during the tester's own QA activities.
