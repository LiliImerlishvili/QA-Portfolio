**Test Case ID:** TC-006

## Objective
Verify that the app does not crash and handles denied camera permission gracefully.

## Preconditions
- User is logged in
- Camera permission is set to Denied in device OS settings
- Chat conversation is opened

## Test Steps
1. Open the media picker/attachment menu
2. Select "Camera"
3. Observe app behavior

## Expected Result
- App does not crash
- User sees permission prompt or guidance message (example: "Enable Camera permission in Settings")
- User can cancel and continue using the app normally

## Postconditions
- App remains stable and user stays on the chat screen
