# 🐞 BUG-008 – Newsletter Opt-in Checkbox State Not Saved

**Priority:** Low  
**Severity:** Minor  

## Description
Newsletter opt-in checkbox state is not saved after form submission.

## Preconditions
- User fills subscription form

## Steps to Reproduce
1. Check newsletter opt-in box
2. Submit the form
3. Reopen profile or form

## Actual Result
Checkbox is unchecked.

## Expected Result
User preference should be saved correctly.
