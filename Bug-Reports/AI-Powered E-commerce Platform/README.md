# AI-Powered E-commerce Platform — Bug Reports

This folder contains NDA-safe bug reports based on real QA findings from an e-commerce platform with integrated AI Chat, AI Search, product discovery, promotions, and administrative functionality.

All sensitive information has been removed or generalized, including:

- Original platform and company names
- Branch and location names
- Website URLs
- Store, brand, and product names
- Tester and employee names
- Internal identifiers and proprietary data
