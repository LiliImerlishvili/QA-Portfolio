**Test Case ID:** TC-002  
**Related Bug:** BUG-002  
**Test Type:** Functional  
**Priority:** Medium  

## Objective
Verify that message status updates correctly after the receiver gets the message.

## Preconditions
- User A and User B are logged in on separate devices/emulators
- A chat conversation exists between User A and User B
- Both devices have stable network connection

## Test Steps
1. On User A device, open chat with User B
2. Send a short text message
3. Observe the message status indicator on User A (Sending/Sent/Delivered)
4. On User B device, confirm the message appears in the chat
5. Return to User A device and observe the status again

## Expected Result
- Status updates from "Sending" to "Sent/Delivered" after the receiver gets the message
- No indefinite "Sending" state remains

## Postconditions
- Message is visible on both devices and status is accurate
