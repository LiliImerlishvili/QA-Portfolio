**Bug ID:** BUG-002  
**Priority:** Medium  
**Severity:** Major  
**Status:** Open  

## Description
The system accepts invalid email formats during user registration, allowing account creation with incorrect email addresses.

## Preconditions
- User is not logged in
- Registration page is accessible

## Steps to Reproduce
1. Open the Registration page
2. Enter invalid email (example: test@)
3. Enter valid password
4. Click "Register"

## Actual Result
Account is created using an invalid email address.

## Expected Result
Email format should be validated and an error message should be displayed.

## Environment
- Browser: Chrome 121
- OS: macOS
- Environment: Staging
