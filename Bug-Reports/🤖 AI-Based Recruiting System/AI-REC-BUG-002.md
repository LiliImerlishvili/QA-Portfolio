# Bug Report AI-REC-BUG-002: User Cannot Access Profile Page

**Project:** AI-Based Recruiting System (Confidential)  
**Severity:** Major  
**Priority:** Urgent  
**Reproducibility:** 80%  

---

## Test Environment

| Item                | Value                                              |
|---------------------|----------------------------------------------------|
| Device              | iMac                                               |
| Operating System    | macOS Tahoe 26.0.1                                 |
| Browser             | Google Chrome Version 141.0.7390.66 (Official Build) (arm64) |

---

## Precondition
Employer user has access to the system and can log in successfully.

---

## Steps to Reproduce

1. Open the web platform (AI recruiting system).  
2. Log in using an employer account.  
3. From the main navigation, click "Settings".  
4. Go to “Account Settings”.  
5. From the displayed menu on the right side, select "Profile".  
6. Attempt to open the Profile page.

---

## Actual Result

Profile page does not open. Instead, the system displays an error message indicating:  
**"User not found"**

---

## Expected Result

The Profile page should load successfully, displaying all employer profile information and general dashboard details.
