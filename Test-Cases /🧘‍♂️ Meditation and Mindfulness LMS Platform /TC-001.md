# 🧪 TC-001 – Verify Access Restriction for Paid Courses

**Related Bug:** BUG-001  
**Priority:** High  
**Test Type:** Functional / Security  

## Preconditions
- User is logged in
- User does NOT have an active subscription
- Paid course exists

## Test Steps
1. Navigate to a paid course page
2. Click on a locked lesson

## Expected Result
User should not be able to access the lesson and should see a subscription prompt.
