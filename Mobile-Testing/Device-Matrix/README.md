# Mobile Device Matrix

This device matrix defines the mobile devices, operating systems, screen sizes, and network conditions used for compatibility testing.

## Device Coverage

- Android smartphones
- iPhones
- Different screen sizes
- Different OS versions
- Wi-Fi and mobile data
- Stable and unstable network conditions

## Matrix Fields

- Device model
- Operating system
- OS version
- Screen size
- Resolution
- Network type
- Test priority
- Notes
