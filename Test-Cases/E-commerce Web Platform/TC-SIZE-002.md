# TC-SIZE-002 – Unavailable Sizes Cannot Be Selected

**Purpose:**  
To ensure unavailable sizes are disabled or removed.

**Related Bug:** UI-SIZE-081  
**Module / Feature:** Size Selector  
**Priority:** Medium  

---

### Steps:
1. Open Size dropdown on product page.  
2. Identify sizes that should not be available.  
3. Try selecting an unavailable size.

---

### Expected Result:
- Unavailable sizes are either hidden or disabled.  
- System does not allow selection of invalid size.
