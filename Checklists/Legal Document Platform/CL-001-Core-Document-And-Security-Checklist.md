# CL-001 — Core Document and Security Checklist

## Login and Accessibility

- [ ] Remember Me is readable
- [ ] Contrast meets accessibility expectations
- [ ] Login does not show false profile warnings

## OTP Security

- [ ] Failed OTP attempts are counted
- [ ] OTP is invalidated after the limit
- [ ] Temporary lock or new OTP is required

## Partner Validation

- [ ] First Name rejects invalid symbols
- [ ] Legal Entity requires Company Name
- [ ] Validation messages are clear

## Input Security

- [ ] HTML is encoded
- [ ] JavaScript is blocked
- [ ] Stored input cannot execute
- [ ] DOM does not contain unsafe markup

## Document Management

- [ ] Permanent deletion removes the document
- [ ] Deleted documents disappear for all users
- [ ] Shared views remain synchronized
- [ ] Georgian text renders correctly
