# 🐞 BUG-010 – Booking session is lost after page refresh

**Priority:** High  
**Severity:** Major  

## Description
Refreshing the page during the booking process breaks the booking session.

## Preconditions
- Booking process has been started

## Steps to Reproduce
1. Start the booking process
2. Refresh the page

## Actual Result
User is redirected to an error page.

## Expected Result
Booking session should be preserved after refresh.
