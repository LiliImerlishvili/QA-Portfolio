**Test Case ID:** TC-005  
**Related Bug:** BUG-005  
**Test Type:** Functional / Performance  
**Priority:** High  

## Objective
Verify that media upload completes, resumes, or fails gracefully when the app is moved to background during upload.

## Preconditions
- User is logged in
- Chat conversation is opened
- Storage/Photos permission is granted
- Stable internet connection is available

## Test Steps
1. Start uploading an image in chat
2. Immediately move the app to background (Home button)
3. Wait 10-15 seconds
4. Return to the app
5. Observe upload state and UI feedback

## Expected Result
- Upload completes successfully OR resumes after returning to the app
- If upload fails, a clear error message is shown with a retry option
- Upload should not remain stuck indefinitely

## Postconditions
- Media message is either successfully sent or clearly marked as failed with recovery options
