# Travel Reservation Mobile Application — Test Cases

This folder contains test cases created for an NDA-protected mobile travel and reservation application.

The test cases are directly linked to the reported defects and cover authentication, registration, hotel discovery, maps, chat attachments, media playback, and account data integrity.

## Test Case Index

| ID | Title | Related Defect | Priority |
|---|---|---|---|
| TC-001 | Validate Invalid Input in Forgot Password Flow | BR-001 | High |
| TC-002 | Verify a Single Verification Code Is Sent per Registration Request | BR-002 | High |
| TC-003 | Verify Chat Document Upload, File Name Display, and Download | BR-003 | High |
| TC-004 | Verify Explore Stays Opens the Hotel Search Flow | BR-004 | Critical |
| TC-005 | Verify Current Location Is Displayed on the Map | BR-005 | High |
| TC-006 | Verify Chat Video Can Be Opened and Played | BR-006 | High |
| TC-007 | Verify Existing Email Is Rejected During Email Change | BR-007 | Critical |

## Coverage

- Forgot password validation
- Email verification

- Hotel discovery navigation
- Location and map behavior
- Chat video playback
- Email uniqueness validation

- Security-related validation
