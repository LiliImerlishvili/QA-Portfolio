# TC-FLTR-002 – Filter Persistence After Multiple Page Navigation

**Purpose:**  
To ensure filters remain applied after moving between pages.

**Related Bug:** UI-FLTR-093  
**Module / Feature:** Filters / Pagination  
**Priority:** High  

---

### Steps:
1. Apply category filter.  
2. Click "View more" or go to next page.  
3. Navigate back and forth between pages.  
4. Open and close multiple products.  
5. Return to listing and review products.

---

### Expected Result:
- Filter remains active.  
- Products still match filter criteria.  

---

### Status:
❌ Failed
