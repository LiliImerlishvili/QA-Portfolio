# 🤖 AI-Powered Multi-Modal Platform

This project represents manual QA work performed on a **multi-modal AI platform**
that combines conversational AI with **audio, image, and video generation features**.
The platform integrates multiple AI models and supports complex, prompt-driven user workflows.

The primary focus of testing was ensuring **AI response consistency, feature stability,
and reliable user experience** across different AI-powered functionalities.

---

## 👩‍💻 Role

**Manual QA Engineer**

---

## 🧪 Responsibilities

- Tested AI-driven conversational chat workflows and prompt-based interactions to ensure consistent responses and reliable user flows  
- Performed functional and exploratory testing of **audio generation features**, validating output accuracy, UI behavior, and edge cases  
- Tested **image generation and image editing** functionality, identifying prompt-handling and rendering issues  
- Validated **video generation flows**, media processing logic, and related UI interactions  
- Tested **AI content detection and plagiarism-related features** across multiple input scenarios  
- Verified integration stability across **multiple AI models** and shared workflows  
- Reported structured bug reports in Jira with clear reproduction steps and supported retesting cycles in collaboration with developers
