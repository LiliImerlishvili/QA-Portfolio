# TC-RE-002 - Verify Listing Card Information

**Priority:** Medium

## Preconditions
- User is on the property listings page
- Listings are displayed as cards

## Steps
1. Review 3 different listing cards
2. Compare visible data between cards (title, price, location, thumbnail)

## Expected Result
- Each listing card shows consistent structure
- Title, price, and location are visible and readable
- No duplicated or missing values are displayed
