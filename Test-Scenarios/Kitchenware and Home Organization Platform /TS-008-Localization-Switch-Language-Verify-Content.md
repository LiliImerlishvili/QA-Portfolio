# TS-008 - Localization: Switch Language and Verify Content

## Goal
Verify that language switching updates UI labels and key content consistently across pages.

## Preconditions
- Website supports at least two languages.
- Language switcher is available in header/footer.

## Flow (High-level)
1. User opens homepage in default language.
2. User switches site language.
3. User navigates to product listing and product detail page.
4. User opens cart and checkout pages.
5. User verifies UI labels and key content reflect the selected language consistently.
