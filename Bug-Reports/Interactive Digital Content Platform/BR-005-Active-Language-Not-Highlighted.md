# BR-005 — Active Language Is Not Visually Highlighted

## Summary

The currently selected language has no clear active-state indicator.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iMac |
| Operating System | macOS Tahoe 26.3.1 |
| Browser | Google Chrome 146 |
| Reproducibility | 100% |

## Severity

Medium

## Priority

High

## Preconditions

- The relevant feature is available.
- The user has the required access level.

## Steps to Reproduce

1. Open the language selector.
2. Select another language.
3. Observe the selected language option.

## Actual Result

The active language is not visually distinguishable from inactive options.

## Expected Result

The selected language should have a clear active state, such as highlight, underline, bold text, or an active icon.

## User or Business Impact

Users cannot easily confirm which language is active.
