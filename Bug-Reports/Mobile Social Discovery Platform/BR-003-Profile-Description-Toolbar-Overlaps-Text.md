# BR-003 — Profile Description Toolbar Overlaps Long Text Content

## Summary

When the profile description contains long text, the action toolbar overlaps the content.

## Environment

| Field | Value |
|---|---|
| Environment | Production |
| Device | iPhone 14 Pro |
| Operating System | iOS 26.5 |
| Application Version | 2.0.0 (build 2003) |
| Reproducibility | 100% |

## Severity

High

## Priority

Normal

## Preconditions

- The user has access to the relevant flow.
- The required feature is available.

## Steps to Reproduce

1. Open the profile screen.
2. Enter edit mode.
3. Open the profile description field.
4. Enter a long multi-line description.
5. Observe the toolbar and text layout.

## Actual Result

The toolbar overlaps the description text and hides part of the content.

## Expected Result

The text field should scroll or resize so the full content remains readable and editable.

## User or Business Impact

Users cannot review or edit long profile descriptions comfortably.
