# 🐞 BUG-002 – Course Progress Not Saved After Page Refresh

**Priority:** Medium  
**Severity:** Major  

## Description
User progress within a meditation lesson is lost after refreshing the page.

## Preconditions
- User is logged in
- User has access to the course
- Lesson video is playing

## Steps to Reproduce
1. Start a lesson video
2. Watch until mid-progress
3. Refresh the page

## Actual Result
Lesson progress resets to the beginning.

## Expected Result
Progress should resume from the last watched timestamp.
