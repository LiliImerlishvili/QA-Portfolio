The system displays a generic error message when login fails, without explaining the cause.

## Preconditions
- User is on login page

2. Enter incorrect password
3. Click "Login"

## Actual Result
Generic error message is displayed.

## Expected Result
Clear error message explaining incorrect credentials should be shown.
