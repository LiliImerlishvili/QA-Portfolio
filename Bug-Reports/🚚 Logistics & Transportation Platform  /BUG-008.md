# 🐞 BUG-008 – Booking API returns success for failed submission

**Priority:** High  
**Severity:** Critical  

## Description
The booking API returns a success response even when the booking fails internally.

## Preconditions
- Invalid booking payload

## Steps to Reproduce
1. Send an invalid booking request
2. Observe the API response

## Actual Result
API responds with success status.

## Expected Result
API should return an appropriate error response.
