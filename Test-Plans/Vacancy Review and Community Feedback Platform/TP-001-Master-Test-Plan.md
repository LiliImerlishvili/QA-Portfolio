# TP-001 — Master Test Plan

## Objective
Define the QA strategy for a vacancy review and community feedback platform.

## Scope
- Vacancy import and categorization
- Company and vacancy reviews
- Comments and community interactions
- Quick Polls and statistics
- Authentication and session management
- Content validation
- Responsive and cross-browser behavior

## Test Types
- Functional testing
- Exploratory testing
- Regression testing
- UI testing
- API and data-consistency testing
- Negative testing
- Cross-browser testing

## Exit Criteria
- Critical and high-severity defects are resolved or formally accepted
- Regression scope is completed
- Core user journeys pass
- Known risks are documented
