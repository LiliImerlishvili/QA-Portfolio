# 🐞 BUG-005 – Language Switch Does Not Update Course Description

**Priority:** Low  
**Severity:** Minor  

## Description
Course description remains in the original language after language switch.

## Preconditions
- Multilingual support enabled
- Course has translated content

## Steps to Reproduce
1. Open a course page
2. Switch site language
3. Observe course description

## Actual Result
Description text remains unchanged.

## Expected Result
All course content should update to selected language.
