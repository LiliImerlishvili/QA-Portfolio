**Test Case ID:** TC-003  
**Related Bug:** BUG-003  
**Test Type:** Functional  
**Priority:** Medium  

## Objective
Verify that the receiver gets only a single push notification for a single incoming message.

## Preconditions
- Push notifications are enabled on User B device
- User B is logged in
- App is running in background on User B device
- User A can send a message to User B

## Test Steps
1. Put the app in background on User B device
2. From User A device, send one message to User B
3. Observe notifications on User B device notification tray

## Expected Result
- Only one push notification is displayed for the single message
- Notification content matches the message sender and context

## Postconditions
- User B has one notification related to the message event
