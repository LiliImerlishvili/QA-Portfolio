# BR — Last Item in Interests Panel Is Partially Cut Off

## Summary

The final item in the Interests panel is partially hidden and not fully readable.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | Android Studio Emulator — Medium Phone API 36.1 |
| Operating System | Android 16 |
| Application Version | 2.0.1 |
| Reproducibility | 100% |

## Severity

Medium

## Priority

Low

## Preconditions

- The user is authenticated unless otherwise stated.
- The relevant feature is available.

## Steps to Reproduce

1. Open the Profile screen.
2. Open Interests.
3. Scroll to the final item.

## Actual Result

The last item is clipped by the panel boundary.

## Expected Result

All interest items should be fully visible and readable.

## User Impact

The interface appears incomplete and unpolished.
