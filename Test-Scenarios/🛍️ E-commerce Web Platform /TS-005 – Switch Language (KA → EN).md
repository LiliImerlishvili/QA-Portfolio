# TS-005 – Switch Language (KA → EN) and Verify Full Translation

**Goal:**  
Validate that language switcher correctly updates all main UI texts, including banners.

**Preconditions:**  
- Website supports KA and EN languages.  

**Flow:**  
1. User opens homepage in KA.  
2. User switches language to EN using the language switcher.  
3. User reviews main sections: header, menus, categories, product cards.  
4. User scrolls and checks Service Info Banner and footer.  
5. User confirms that no Georgian text remains.
