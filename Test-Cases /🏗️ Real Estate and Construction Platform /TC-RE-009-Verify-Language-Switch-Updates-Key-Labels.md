# TC-RE-009 - Language Switch Updates Key Labels

**Priority:** Low

## Preconditions
- Platform supports at least two languages
- Language switcher is visible

## Steps
1. Switch site language
2. Open listings page and property details page
3. Check key UI labels (navigation, CTAs, form labels)

## Expected Result
- Selected language is applied consistently
- No mixed-language labels appear on key pages
- Form labels and CTA text match the selected language
