# 🐞 BUG-006 – AI Prediction Scan Fails for Non-English Text (“Cannot Create Answer”)

---

## 📌 Summary
AI Prediction feature fails when scanning **non-English text**, returning a **“Cannot Create Answer”** error instead of processing the input and providing a prediction result.

---

## 🖥 Environment

| Parameter | Value |
|---------|-------|
| Platform | Web |
| Device | iMac |
| Browser | Google Chrome 143.0.7499.193 (Official Build, arm64) |
| OS | iOS Tahoe 26.2 |
| Reproducibility | Always (100%) |

---

## 🏷 Labels

- Web  
- AI Prediction  
- Localization  
- Multilingual  

---

## 📝 Description

When scanning text written in a language other than English (for example Georgian, Turkish, or Bemba), the **AI Prediction** feature fails to process the input.

After clicking the **Scan** button, the system displays the error message **“Cannot Create Answer”**, and no prediction result is returned, despite valid user state and sufficient credits.

This indicates that the AI Prediction feature does not properly support or handle non-English input.

---

## 🔁 Preconditions

- User is logged in
- User has an active **Silver subscription**
- User has sufficient credits to use **AI Prediction**

---

## 🧪 Steps to Reproduce

1. Navigate to the **AI Prediction** section  
2. Paste text written in a **non-English language** (e.g. Georgian, Turkish, Bemba)  
3. Enter a valid value in the **Title** field  
4. Click the **Scan** button  
5. Observe the system behavior  

---

## ❌ Actual Result

- Scan fails
- Error message **“Cannot Create Answer”** is displayed
- No prediction result is shown

---

## ✅ Expected Result

- System processes **non-English text** correctly
- AI Prediction generates a valid result regardless of input language
- No error message is displayed for supported languages

---

## 📉 Impact

- AI Prediction feature is unusable for non-English users
- Limits platform accessibility and multilingual support
- Reduces trust in AI capabilities for international users

---

## 🔄 Frequency

- Always (100%)

---

## ⚠ Severity

- Medium

---

## 🚦 Priority

- Medium


---

## 📝 Notes

- Issue occurs consistently across different non-English languages
- Not related to user permissions or insufficient credits
- Suggests missing language handling or model limitation

---

## 💡 Recommendation

Ensure AI Prediction supports multilingual input or clearly communicates supported languages.  
Implement proper language detection, fallback handling, or user guidance to prevent generic system errors.
