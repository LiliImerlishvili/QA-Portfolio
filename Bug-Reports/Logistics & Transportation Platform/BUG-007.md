# 🐞 BUG-007 – Delivery status labels are not fully localized

**Priority:** Medium  
**Severity:** Minor  

## Description
Some delivery status labels remain untranslated after language switch.

## Preconditions
- Localization is enabled

## Steps to Reproduce
1. Switch the application language
2. Open delivery tracking page

## Actual Result
Some labels appear in the default language.

## Expected Result
All delivery status labels should be localized.
