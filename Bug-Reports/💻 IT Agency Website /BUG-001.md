**Bug ID:** BUG-001  
**Priority:** High  

## Description
Homepage slider animation causes layout shift on tablet resolutions, overlapping content below the slider.

## Preconditions
- Tablet viewport enabled
- Homepage loaded

## Steps to Reproduce
1. Open homepage on tablet device or emulator
2. Observe slider animation on page load

## Actual Result
Slider animation pushes content down and overlaps sections.

## Expected Result
Slider animation should not affect page layout or overlap content.
