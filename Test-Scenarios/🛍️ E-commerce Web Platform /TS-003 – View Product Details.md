# TS-003 – View Product Details, Select Size, and Open Size Chart

**Goal:**  
Check that user can view correct product details, choose size, and see correct size chart.

**Preconditions:**  
- Product has multiple sizes configured.  
- Size guide is available.

**Flow:**  
1. User opens a product page.  
2. User reviews product title, brand, attributes.  
3. User opens Size dropdown and checks available sizes.  
4. User clicks “Size Guide” / Size Chart.  
5. User verifies that chart information belongs to the current product.
