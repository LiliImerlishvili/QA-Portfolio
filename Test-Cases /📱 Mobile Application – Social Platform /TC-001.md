**Test Case ID:** TC-001  
**Related Bug:** BUG-001  
**Test Type:** Functional / UI  
**Priority:** High  

## Objective
Verify that the app blocks sending chat messages that contain only whitespace characters.

## Preconditions
- User is logged in
- A chat conversation is available and opened

## Test Steps
1. Tap the message input field
2. Enter only spaces (example: "     ")
3. Tap the "Send" button

## Expected Result
- The message is not sent
- A validation hint is displayed (example: "Message cannot be empty")

## Postconditions
- No new message appears in the chat thread
