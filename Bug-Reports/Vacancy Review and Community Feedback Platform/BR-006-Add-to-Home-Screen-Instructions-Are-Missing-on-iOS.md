# BR-006 — Add to Home Screen Instructions Are Missing on iOS

## Summary

After selecting Add to Home Screen on iOS, the application does not show the expected installation guidance.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iMac |
| Operating System | macOS Tahoe 26.5.1 |
| Browser | Google Chrome 149 |
| Reproducibility | 100% |

## Severity

High

## Priority

High

## Preconditions

- The user has access to the relevant flow.
- The required platform data is available.

## Steps to Reproduce

1. Open the website on iOS Safari.
2. Open the browser share menu.
3. Select Add to Home Screen.
4. Observe the result.

## Actual Result

No installation instructions or guidance are shown.

## Expected Result

The user should receive clear instructions and confirmation for adding the application to the Home Screen.

## User or Business Impact

Users may not understand how to install or launch the web application.
