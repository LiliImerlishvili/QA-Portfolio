**Bug ID:** KHP-BUG-001  
**Priority:** High  

## Description
On mobile devices, the "Add to Cart" button becomes disabled after selecting product variants, preventing users from adding the product to the cart.

## Preconditions
- User is on a product detail page
- Product has multiple variants (size, color)
- User is using a mobile browser

## Steps to Reproduce
1. Open a product detail page
2. Select a color variant
3. Select a size variant
4. Observe the "Add to Cart" button

## Actual Result
The "Add to Cart" button becomes inactive and cannot be clicked.

## Expected Result
The "Add to Cart" button remains active and the product can be added to the cart.
