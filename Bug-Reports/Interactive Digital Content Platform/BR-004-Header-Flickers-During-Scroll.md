# BR-004 — Header Flickers During Scroll on Content Pages

## Summary

The website header visibly flickers while the user scrolls on story, educational, or digital-content pages.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iMac |
| Operating System | macOS Tahoe 26.3.1 |
| Browser | Google Chrome 146 |
| Reproducibility | 100% |

## Severity

Medium

## Priority

Normal

## Preconditions

- The relevant feature is available.
- The user has the required access level.

## Steps to Reproduce

1. Open the platform.
2. Navigate to a content category page.
3. Scroll up and down repeatedly.

## Actual Result

The header flickers or becomes visually unstable during scrolling.

## Expected Result

The header should remain stable and render smoothly while scrolling.

## User or Business Impact

The issue creates visual distraction and reduces perceived quality.
