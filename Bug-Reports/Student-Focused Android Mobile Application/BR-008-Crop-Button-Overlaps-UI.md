# BR — Image Crop Button Overlaps Other UI Elements

## Summary

The Crop button overlaps or appears mixed with other controls during image upload.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | Android Studio Emulator — Medium Phone API 36.1 |
| Operating System | Android 16 |
| Application Version | 2.0.1 |
| Reproducibility | 100% |

## Severity

Medium

## Priority

Normal

## Preconditions

- The user is authenticated unless otherwise stated.
- The relevant feature is available.

## Steps to Reproduce

1. Open Activity.
2. Tap Add.
3. Choose an image.
4. Open crop mode.
5. Observe the layout.

## Actual Result

The Crop button overlaps other elements or appears incorrectly positioned.

## Expected Result

The Crop button should remain clearly visible and separated from other controls.

## User Impact

Image editing becomes confusing and difficult.

## Attachment

Evidence available in the original defect management system.
