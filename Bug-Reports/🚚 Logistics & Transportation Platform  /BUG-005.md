# 🐞 BUG-005 – Booking data is cleared after language switch

**Priority:** Medium  
**Severity:** Major  

## Description
Switching the application language clears previously entered booking data.

## Preconditions
- Booking form is partially completed

## Steps to Reproduce
1. Enter booking details
2. Change the application language

## Actual Result
All entered data is cleared.

## Expected Result
Entered booking data should be preserved.
