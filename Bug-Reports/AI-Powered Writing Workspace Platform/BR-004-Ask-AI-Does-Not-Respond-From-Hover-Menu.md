# BR-004 — Ask AI Does Not Respond From the Screenplay Hover Menu

## Summary

The Ask AI action does not respond when launched from the screenplay hover menu.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iMac |
| Operating System | macOS Tahoe 26.4.1 |
| Browser | Google Chrome 147 |
| Reproducibility | 100% |

## Severity

High

## Priority

Urgent

## Preconditions

- The user is authenticated.
- The relevant project or draft exists.
- The required feature is available.

## Steps to Reproduce

1. Open a screenplay draft.
2. Select text.
3. Open the contextual hover menu.
4. Choose Ask AI.
5. Observe the response.

## Actual Result

The Ask AI action does not open the AI panel or process the request.

## Expected Result

The AI panel should open and process the selected text request.

## User or Business Impact

Users cannot use AI assistance from the contextual editing flow.
