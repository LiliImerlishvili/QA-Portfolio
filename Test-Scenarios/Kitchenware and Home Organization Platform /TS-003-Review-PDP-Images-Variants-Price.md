# TS-003 - Review Product Details, Images, Variants, and Price

## Goal
Verify that product detail page displays correct information and variant selection updates UI consistently.

## Preconditions
- Website is available.
- Product has images and at least one variant (size/color) if applicable.
- Product is visible in listing and accessible via URL.

## Flow (High-level)
1. User opens a product detail page (PDP).
2. User reviews title, price, description, and availability.
3. User interacts with product gallery (switch images).
4. User selects available variants (size/color).
5. User verifies price and availability reflect the selected variant.
