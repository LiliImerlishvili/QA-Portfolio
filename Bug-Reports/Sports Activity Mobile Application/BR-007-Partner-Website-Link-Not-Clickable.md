# BR-007 — Featured Partner Website Link Is Not Clickable

## Summary

The partner website URL is shown but cannot be tapped.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iPhone 14 Pro |
| Operating System | iOS 26.4.2 |
| Application Version | 1.0 (build 7) |
| Reproducibility | 100% |

## Severity

High

## Priority

High

## Preconditions

- The user is authenticated unless otherwise stated.
- The relevant feature is available.

## Steps to Reproduce

1. Open the Featured section.
2. Open a partner details modal.
3. Tap the company website link.

## Actual Result

Nothing happens when the link is tapped.

## Expected Result

The website should open in an in-app or external browser.

## User Impact

Users cannot access partner websites.

## Attachment

Evidence available in the original defect management system.
