# 🐞 BUG-003 – Booking form allows submission with empty required fields

**Priority:** High  
**Severity:** Critical  

## Description
The booking form allows submission even when mandatory fields are empty.

## Preconditions
- User is on booking page

## Steps to Reproduce
1. Leave all required fields empty
2. Click the "Submit Booking" button

## Actual Result
The booking is submitted successfully.

## Expected Result
Submission should be blocked and validation errors displayed.
