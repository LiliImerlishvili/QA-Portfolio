# BR-009 — Georgian Font Is Rendered Incorrectly in Generated Document

## Summary

Georgian text in generated documents is displayed with an incorrect font or broken rendering.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iMac |
| Operating System | macOS Tahoe 26.3.1 |
| Browser | Google Chrome 146 |
| Reproducibility | 100% |

## Severity

High

## Priority

Urgent

## Preconditions

- The user is authenticated unless otherwise stated.
- The relevant feature is available.

## Steps to Reproduce

1. Open document generation.
2. Enter Georgian text.
3. Generate the document.
4. Review the output.

## Actual Result

Georgian characters are rendered with the wrong font or broken visual output.

## Expected Result

Georgian text should use a compatible font and render correctly.

## User or Business Impact

Generated legal documents may become unreadable or visually invalid.

## Attachment

Evidence available in the original defect management system.
