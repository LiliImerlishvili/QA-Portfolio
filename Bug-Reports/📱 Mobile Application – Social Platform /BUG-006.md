**Bug ID:** BUG-006  
**Priority:** High  
**Severity:** Critical  
**Status:** Open  

## Description
The app crashes when the user tries to open the camera from the media picker while camera permission is denied.

## Preconditions
- User is logged in
- Camera permission is set to Denied in OS settings
- Chat screen is opened

## Steps to Reproduce
1. Open chat conversation
2. Tap attachment/media icon
3. Tap "Camera"
4. Observe app behavior

## Actual Result
App crashes or closes unexpectedly.

## Expected Result
App should show permission prompt or an informative message with a path to Settings without crashing.

## Environment
- Device: Android Emulator (Pixel 5)
- OS: Android 12
- App Build: Staging
- Network: Wi-Fi
