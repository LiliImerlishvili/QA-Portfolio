# TS-002 - Search Product and Open Product Detail Page

## Goal
Verify that search returns relevant results and user can open a product from search results.

## Preconditions
- Website is available.
- Search bar is visible and enabled.
- At least one searchable product exists in the catalog.

## Flow (High-level)
1. User enters a product keyword in the search bar.
2. User submits the search query.
3. User reviews the results list.
4. User opens a product from the results.
5. User verifies product detail page loads and matches the selected result.
