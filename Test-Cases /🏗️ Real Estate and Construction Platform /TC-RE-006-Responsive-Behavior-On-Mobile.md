# TC-RE-006 - Responsive Behavior on Mobile

**Priority:** High

## Preconditions
- Mobile device or emulator is used
- User opens a property details page

## Steps
1. Scroll through the full page
2. Interact with main CTA buttons (Inquiry / Contact)
3. Open and close any expandable sections (if available)

## Expected Result
- No horizontal scrolling appears
- Text, images, and buttons remain aligned
- CTAs are clickable and not overlapped by UI elements
