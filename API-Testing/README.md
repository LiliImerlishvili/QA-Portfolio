# API Testing

This folder contains API testing documentation, request collections, validations, negative scenarios, and supporting materials.

## Coverage

- Request methods and endpoints
- Authentication
- Headers and parameters
- Request body validation
- Response status validation
- JSON schema validation
- Positive and negative scenarios
- Boundary and missing-field checks
- Data-driven testing
- Error handling
- Environment variables
- Collection execution

## Tools

- Postman
- Newman
- JavaScript test scripts
- JSON and CSV data files
- GitHub

## Confidentiality

Real tokens, credentials, private endpoints, and sensitive payloads are not included.
