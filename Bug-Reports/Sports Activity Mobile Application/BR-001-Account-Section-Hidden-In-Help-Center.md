# BR-001 — Account Section Is Hidden in Help Center After Expanding Other Categories

## Summary

The Account section is no longer visible after expanding the Pearcoin and Marketplace sections in Help Center.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iPhone 14 Pro |
| Operating System | iOS 26.4.2 |
| Application Version | 1.0 (build 7) |
| Reproducibility | 100% |

## Severity

Medium

## Priority

Normal

## Preconditions

- The user is authenticated unless otherwise stated.
- The relevant feature is available.

## Steps to Reproduce

1. Open Help Center.
2. Expand Pearcoin.
3. Expand Marketplace.
4. Scroll to the bottom of the page.

## Actual Result

The Account section is hidden and cannot be accessed.

## Expected Result

All Help Center sections should remain visible and scrollable after expanding other sections.

## User Impact

Users cannot access account-related help content.

## Attachment

Evidence available in the original defect management system.
