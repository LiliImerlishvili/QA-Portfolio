# TS-001 - Browse Categories, Apply Filters, Open Product

## Goal
Verify that user can browse categories, apply filters, and open a relevant product.

## Preconditions
- Website is available.
- Product catalog contains multiple categories and products.
- Filters and sorting are enabled on listing pages.

## Flow (High-level)
1. User opens homepage.
2. User navigates to a main category and then a subcategory.
3. User applies one or more filters (brand, price range, material).
4. User optionally applies sorting (e.g., price low to high).
5. User opens one of the filtered products.
6. User verifies the opened product matches selected category/filter context.
