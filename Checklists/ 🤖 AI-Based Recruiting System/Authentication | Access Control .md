## 9. Role-Based Access Control Checklist

**ID:** AI-REC-CL-009  
**Module:** Authentication / Access Control

- [ ] Admin can access all restricted modules  
- [ ] HR can access HR-related views  
- [ ] Basic users cannot access admin modules  
- [ ] Unauthorized users receive proper access errors  
- [ ] No restricted module loads accidentally  
- [ ] All permission checks work correctly
