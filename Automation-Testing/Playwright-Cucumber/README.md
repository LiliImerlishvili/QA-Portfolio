# Playwright and Cucumber Automation

This section contains UI automation examples created with Playwright, Cucumber.js, and TypeScript.

## Technologies

- Playwright
- Cucumber.js
- TypeScript
- Node.js

## Covered Areas

- Homepage validation
- Header and navigation testing
- Dropdown menu testing
- Login and authentication testing
- OTP validation
- Positive and negative test scenarios

## Framework Structure

- Feature files written in Gherkin
- Step definitions
- Page Object Model
- Hooks and test setup
- Reusable assertions and helper methods
