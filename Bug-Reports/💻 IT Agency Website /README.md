##  IT Agency Website (NDA Project)

**Role:** Manual QA Engineer  
**Period:** 2025  

###  Project Overview
Manual QA testing for a corporate IT agency website focused on lead generation, multilingual content, UI/UX consistency, analytics accuracy, and cross-device performance.

###  QA Responsibilities
- Perform end-to-end testing across desktop, tablet, and mobile
- Validate navigation structure, multilingual flows, and contact forms
- Test lead-generation forms and CTA behavior
- Identify UI/UX issues in sliders, animations, and interactive elements
- Conduct cross-browser and responsive testing
- Verify cookie consent behavior and GA4 analytics events
- Report defects and support retesting before releases
