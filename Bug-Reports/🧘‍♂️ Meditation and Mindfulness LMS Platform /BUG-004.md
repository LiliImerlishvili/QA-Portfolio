# 🐞 BUG-004 – PayPal Button Not Clickable on Mobile Devices

**Priority:** Medium  
**Severity:** Major  

## Description
PayPal payment button is not clickable on mobile view.

## Preconditions
- User is on mobile device
- PayPal payment option available

## Steps to Reproduce
1. Open checkout page on mobile
2. Select PayPal option
3. Attempt to click PayPal button

## Actual Result
Button does not respond to user interaction.

## Expected Result
PayPal button should be fully functional on mobile.
