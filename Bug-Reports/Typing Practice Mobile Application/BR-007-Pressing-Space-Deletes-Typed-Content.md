# BR — Pressing Space Deletes Previously Typed Content

## Summary

Pressing the Space key during typing can delete or clear previously entered content.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iPhone 14 Pro |
| Operating System | iOS 26.4.1 |
| Application Version | 2.0.0 (1.1.12) |
| Reproducibility | 100% |

## Severity

Critical

## Priority

Urgent

## Preconditions

- A typing exercise is active.

## Steps to Reproduce

1. Open the application.
2. Start a typing exercise.
3. Type the first word correctly.
4. Press Space once.

## Actual Result

Previously entered text is deleted or the typed content disappears from the input area.

## Expected Result

Pressing Space should add a single space and move the typing flow to the next word without deleting previous content.

## User Impact

Users cannot complete exercises reliably and scoring becomes invalid.

Evidence available in the original defect management system.
