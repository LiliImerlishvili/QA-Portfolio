# TS-001 â€“ Browse Categories, Apply Filters, Open Product

**Goal:**  
Verify that user can browse categories, apply filters, and open a relevant product.

**Preconditions:**  
- Website is available.  
- Product catalog contains multiple categories.

**Flow (High-level):**  
1. User opens homepage.  
2. User navigates to a main category (e.g. Men / Women).  
3. User applies filter (e.g. Shoes, Accessories, Brand X).  
4. User scrolls and opens one of the filtered products.  
5. User verifies that the opened product matches selected category/filter.
