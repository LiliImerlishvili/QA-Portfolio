**Priority:** High

## Description
Inquiry form does not submit when all required fields are filled correctly.

## Preconditions
- Website is live
- User is on Property Details page

## Steps to Reproduce
1. Open a property details page
2. Scroll to inquiry form
3. Fill in all mandatory fields
4. Click "Send Inquiry"

## Actual Result
Form does not submit and no confirmation message is shown.

## Expected Result
Inquiry should be submitted successfully and confirmation message displayed.
