# 🐞 BUG-010 – Logout Redirects User to Blank Page

**Priority:** Medium  
**Severity:** Major  

## Description
User is redirected to a blank page after logging out.

## Preconditions
- User is logged in

## Steps to Reproduce
1. Click Logout button
2. Observe redirect behavior

## Actual Result
Blank page is displayed.

## Expected Result
User should be redirected to homepage or login page.
