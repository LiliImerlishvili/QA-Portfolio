# BR-006 — Insights Detail Screen Displays Glitched Content and No Result

## Summary

Opening a detailed insight displays glitched text and animation instead of the expected insight content.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iPhone 14 Pro |
| Operating System | iOS 26.4.1 |
| Application Version | 2.0.0 (1.1.12) |
| Reproducibility | 100% |

## Severity

High

## Priority

Urgent

## Preconditions

- The user is logged in.
- The Insights section is available.

## Steps to Reproduce

1. Open the application.
2. Navigate to Insights.
3. Open Personalization.
4. Select a detailed insight option.
5. Observe the result screen.

## Actual Result

The screen displays glitched text or animation and no usable insight content.

## Expected Result

The selected insight should load with stable UI, readable text, and the correct personalized result.

## User Impact

A key self-discovery feature becomes unusable.

## Attachment

Evidence available in the original defect management system.
