# BR-006 — Voucher CTA Styling Is Inconsistent Between Mobile Card and Popup

## Summary

The Download Voucher CTA uses inconsistent colors and styling between the mobile card and the voucher popup.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Platform | Responsive Web |
| Device | Mobile |
| Reproducibility | 100% |

## Severity

Medium

## Priority

Normal

## Preconditions

- Voucher download is enabled.
- Both card CTA and popup CTA are visible.

## Steps to Reproduce

1. Open a mobile campaign card.
2. Review the Download Voucher CTA.
3. Open the voucher popup.
4. Compare CTA colors, border, typography, and state behavior.

## Actual Result

The two CTAs use inconsistent visual styling.

## Expected Result

The CTA should use the same approved style in both locations.

## User Impact

The flow appears inconsistent and may confuse users.

Evidence available in the original QA materials.
