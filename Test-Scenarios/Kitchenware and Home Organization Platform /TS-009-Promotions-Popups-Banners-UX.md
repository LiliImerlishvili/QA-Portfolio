# TS-009 - Promotions: Popups, Banners, and UX Behavior

## Goal
Verify that promotional elements (popups, banners) do not block core user journeys and behave correctly on different screens.

## Preconditions
- Promotional popup/banner is enabled (or test environment shows it).
- Website is accessible on mobile and desktop.

## Flow (High-level)
1. User opens homepage and observes promotional elements.
2. User closes popup (if displayed) and continues browsing.
3. User clicks a banner/CTA and verifies navigation is correct.
4. User repeats on mobile screen size.
5. User verifies promotions do not overlap essential UI (navigation, footer, CTAs).
