# BR-007 — Workspace Tab Requires Two Clicks Instead of One

## Summary

Opening a workspace tab requires two clicks.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iMac |
| Operating System | macOS Tahoe 26.4.1 |
| Browser | Google Chrome 147 |
| Reproducibility | 100% |

## Severity

Medium

## Priority

High

## Preconditions

- The user is authenticated.
- The relevant project or draft exists.
- The required feature is available.

## Steps to Reproduce

1. Open a project containing multiple tabs.
2. Click an inactive tab once.
3. Observe whether the tab opens.
4. Click the same tab again.

## Actual Result

The tab opens only after the second click.

## Expected Result

The tab should activate with a single click.

## User or Business Impact

Navigation feels unresponsive and slows down the writing workflow.
