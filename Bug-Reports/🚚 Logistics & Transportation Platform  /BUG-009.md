# 🐞 BUG-009 – Booking page layout breaks on Safari

**Priority:** Medium  
**Severity:** Major  

## Description
UI elements overlap on the booking page when opened in Safari.

## Preconditions
- Safari browser

## Steps to Reproduce
1. Open the booking page in Safari

## Actual Result
Form elements overlap and appear misaligned.

## Expected Result
Layout should be consistent across all supported browsers.
