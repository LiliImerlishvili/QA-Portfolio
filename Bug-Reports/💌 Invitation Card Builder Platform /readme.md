**Role:** Manual QA Engineer

###  Project Overview
Manual QA testing for an interactive web platform that allows users to create, customize, and send digital invitation cards.

The platform focuses on template rendering, image uploads, customization flows, email delivery, and cross-browser UI consistency.

###  QA Responsibilities
- Validate invitation template rendering and customization options
- Test image upload, preview, and placement functionality
- Verify invitation sending and email delivery flow
- Perform cross-browser UI and layout testing
- Identify usability and visual inconsistencies
- Report defects and support retesting cycles
