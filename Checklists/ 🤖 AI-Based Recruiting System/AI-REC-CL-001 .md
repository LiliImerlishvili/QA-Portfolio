## 1. Bulk Import Functionality Checklist

**ID:** AI-REC-CL-001  
**Module:** Bulk Import

- [ ] System accepts valid Excel template  
- [ ] Template structure is validated before processing  
- [ ] Required fields (name, email, position) are checked

- [ ] Validation errors are clear and readable

- [ ] Import completes without partial failures  
- [ ] Successfully imported candidates appear in project  
- [ ] Failed rows are displayed with reasons
