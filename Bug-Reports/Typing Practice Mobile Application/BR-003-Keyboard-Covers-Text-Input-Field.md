# BR — Keyboard Covers the Text Input Field During Practice

## Summary

The on-screen keyboard overlaps the active input area after the user types several characters.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iPhone 14 Pro |
| Operating System | iOS 26.4.1 |
| Application Version | 2.0.0 (1.1.12) |
| Reproducibility | 100% |

## Severity

High

## Priority

Urgent

## Preconditions

- The typing exercise screen is open.

## Steps to Reproduce

1. Open the application.
2. Select a typing exercise.
3. Tap the text input field.
4. Start typing.
5. Continue until the input area moves lower on the screen.

## Actual Result

The keyboard covers the input field, preventing the user from seeing what is being typed.

## Expected Result

The layout should resize or scroll so that the active input field remains visible above the keyboard at all times.

## User Impact

The user cannot monitor typed content, which directly affects accuracy and usability.

Evidence available in the original defect management system.
