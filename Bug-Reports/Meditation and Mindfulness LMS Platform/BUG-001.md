# BUG-001 – Paid course accessible without purchase

**Priority:** High  
**Severity:** Critical  

## Description
Paid meditation course content is accessible without completing checkout.

## Preconditions
- User is logged in
- Course marked as paid

## Steps to Reproduce
1. Open paid course URL directly
2. Access lesson content

## Actual Result
Lesson content is accessible without payment.

## Expected Result
User should be redirected to checkout page.

## Environment
Web / Chrome / Desktop
