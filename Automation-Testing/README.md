# Automation Testing

This folder contains UI automation work and framework documentation.

## Current Focus

- Playwright
- Cucumber
- TypeScript
- Page Object Model
- Reusable step definitions
- Hooks and test setup
- Tags and smoke suites
- Authentication and homepage coverage
- Negative scenarios
- Reporting and CI-ready structure

## Goals

- Reduce repetitive regression effort
- Create stable and maintainable automated checks
- Keep scenarios readable for technical and non-technical stakeholders
- Support future CI execution

## Confidentiality

Automation examples exclude credentials, private URLs, and confidential production data.
