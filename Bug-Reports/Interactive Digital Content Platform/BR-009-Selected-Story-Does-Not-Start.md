# BR-009 — Selected Story Does Not Start Correctly

## Summary

After selecting a story and starting the experience, the story does not launch correctly or remains stuck.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Device | iMac |
| Operating System | macOS Tahoe 26.3.1 |
| Browser | Google Chrome 146 |
| Reproducibility | 100% |

## Severity

High

## Priority

High

## Preconditions

- The relevant feature is available.
- The user has the required access level.

## Steps to Reproduce

1. Open the story catalog.
2. Select an available story.
3. Start the story.
4. Observe the loading and playback behavior.

## Actual Result

The story does not start correctly or the user remains on an incomplete loading state.

## Expected Result

The selected story should load and start consistently.

## User or Business Impact

Users cannot access the main content of the platform.

Evidence is available in the original defect management system. Sensitive URLs and account data are intentionally omitted.
