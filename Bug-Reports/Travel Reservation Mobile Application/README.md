# Travel Reservation Mobile Application — Bug Reports

This folder contains critical and high-impact defects identified during testing of an NDA-protected mobile travel and reservation application.

The application includes:

- User registration and authentication
- Password recovery
- Hotel search and reservation
- Restaurant discovery
- Points of interest search
- Map and location-based features
- Chat and media sharing
- Account and profile management

## Bug Report Index

| ID | Title | Severity | Priority |
|---|---|---|---|
| BR-001 | No Validation Feedback for Invalid Email or Phone Number on Forgot Password | High | Urgent |
| BR-002 | Multiple Verification Codes Are Sent During a Single Registration Attempt | High | High |
| BR-003 | Chat Attachment Displays an Incorrect File Name and Cannot Be Downloaded | High | High |
| BR-004 | Explore Stays Section Is Unresponsive | Critical | Urgent |
| BR-005 | Current Location Is Not Displayed on the Map | High | High |
| BR-006 | Chat Video Cannot Be Opened or Played | High | High |
| BR-007 | Existing Email Address Is Accepted During Email Change | Critical | Urgent |

## Testing Areas Covered

- Authentication and account recovery
- Registration and email verification

- Error handling and user feedback
