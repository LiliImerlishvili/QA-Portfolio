**Bug ID:** BUG-008  
**Priority:** Low

## Description
User session expires without any warning, causing unexpected logout.

## Preconditions
- User is logged in
- Session timeout is near expiry

## Steps to Reproduce
1. Stay inactive for session duration
2. Perform any action

## Actual Result
User is redirected to login page.

## Expected Result
User should receive a session timeout warning.

## Environment
- Browser: Chrome 121
- OS: macOS
- Environment: Staging
