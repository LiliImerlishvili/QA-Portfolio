**Goal:**  
Validate that mobile user can comfortably use the website on smaller resolutions.

**Preconditions:**  
- Mobile device or emulation is available.

**Flow:**  
1. User opens homepage on mobile (or 375px width in DevTools).  
2. User opens menu and navigates to product category.  
3. User scrolls product listing and verifies there is no overlap or broken layout.  
4. User opens product page and checks visibility of title, price, size, and buttons.  
5. User opens Similar Products and scrolls down.
