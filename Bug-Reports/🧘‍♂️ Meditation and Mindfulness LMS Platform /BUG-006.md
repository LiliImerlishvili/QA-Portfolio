# 🐞 BUG-006 – GDPR Consent Banner Reappears After Acceptance

**Priority:** Medium  
**Severity:** Major  

## Description
GDPR cookie consent banner reappears after user has already accepted it.

## Preconditions
- User has accepted cookie consent
- Cookies enabled in browser

## Steps to Reproduce
1. Accept GDPR consent banner
2. Navigate to another page
3. Refresh the browser

## Actual Result
Consent banner is displayed again.

## Expected Result
Banner should not reappear after acceptance.
