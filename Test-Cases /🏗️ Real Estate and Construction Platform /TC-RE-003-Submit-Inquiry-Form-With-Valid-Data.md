# TC-RE-003 - Submit Inquiry Form With Valid Data

**Priority:** High

## Preconditions
- User is on a property details page
- Inquiry form is visible
- Required fields are enabled

## Steps
1. Fill in all required fields with valid data
2. Click "Send Inquiry"
3. Wait for submission response

## Expected Result
- Inquiry is submitted successfully
- User sees confirmation message (success state)
- No infinite loading or form freeze occurs
