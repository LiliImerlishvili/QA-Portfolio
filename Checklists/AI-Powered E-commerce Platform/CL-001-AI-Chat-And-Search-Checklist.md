# CL-001 — AI Chat and Search Checklist

## Language and Conversation

- [ ] Response language matches the user's language
- [ ] Follow-up messages preserve context
- [ ] Escalation messages use the same language
- [ ] No unexpected language switching occurs

## Store and Brand Discovery

- [ ] Store lists are complete
- [ ] Brand lists are complete
- [ ] Responses match the webpage
- [ ] Partial answers are clearly labeled
- [ ] Totals are internally consistent

## Product Availability

- [ ] Available products are reported as available
- [ ] Unavailable products are reported as unavailable
- [ ] Multi-product queries return separate accurate statuses
- [ ] Category-level answers match catalog data

## Offers and Promotions

- [ ] General offers queries return the full list or a direct link
- [ ] Active promotions match the website
- [ ] Expired offers are excluded
- [ ] Partial lists are clearly identified

## Reliability

- [ ] Responses do not contradict earlier messages
- [ ] Catalog-grounded answers remain consistent
- [ ] Missing information is acknowledged clearly
- [ ] Operator escalation is offered appropriately
