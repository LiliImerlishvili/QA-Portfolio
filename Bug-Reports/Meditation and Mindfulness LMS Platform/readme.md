# 🧘‍♂️ Meditation and Mindfulness LMS Platform (NDA Project)

This project represents manual QA work performed on a **Meditation and Mindfulness Learning Management System (LMS)** focused on delivering structured online courses, secure payments, multilingual support, and privacy-compliant user flows.

The platform combines e-learning functionality with e-commerce features, requiring high reliability across learning content access, payment processing, and user data protection.

---

## 👩‍💻 Role

**Manual QA Engineer**

---

## 📅 Period

**September 2025**

---

## 🧪 Responsibilities

- Tested end-to-end learning flows including course access, gated content, and lesson progression  
- Validated e-commerce checkout flows via WooCommerce (Stripe, PayPal, Klarna, Sofort)  
- Verified multilingual behavior across course content and system messages  
- Ensured GDPR compliance, cookie consent enforcement, and secure form submissions  
- Performed cross-browser and responsive testing on desktop, tablet, and mobile devices  
- Validated SEO metadata, GA4 events, and newsletter opt-in tracking accuracy  
- Reported defects in Jira and supported retesting during regression cycles
