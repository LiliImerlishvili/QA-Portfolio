# TC-RE-005 - Image Gallery: Open and Navigate

**Priority:** Medium

## Preconditions
- Property has multiple images
- User is on a property details page

## Steps
1. Click on the main image to open gallery/lightbox
2. Navigate to next and previous images
3. Close the gallery

## Expected Result
- Gallery opens successfully
- Navigation controls work correctly
- Images load without blank placeholders
- Gallery closes and returns user to the same page position
