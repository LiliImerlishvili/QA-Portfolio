# BR-003 — Running Line Icon Selection Does Not Update the Selected Icon

## Summary

Selecting a different running line icon in the editor does not update the icon in the preview or published campaign.

## Environment

| Field | Value |
|---|---|
| Environment | Staging |
| Platform | Web |
| Reproducibility | 100% |

## Severity

Medium

## Priority

High

## Preconditions

- Running line is enabled.
- Icon selection is available.

## Steps to Reproduce

1. Open the campaign editor.
2. Open the running line settings.
3. Select a different icon.
4. Save the changes.
5. Review the preview and published output.

## Actual Result

The icon remains unchanged or does not appear.

## Expected Result

The selected icon should appear immediately in preview and after saving.

## User Impact

Campaign owners cannot customize the promotional message as intended.

## Attachment

Evidence available in the original QA materials.
