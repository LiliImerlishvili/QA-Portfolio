**Bug ID:** BUG-003  
**Priority:** Medium  
**Severity:** Major  
**Status:** Open  

## Description
The receiver sometimes gets two push notifications for the same incoming message. This causes distraction and makes notification counts inaccurate.

## Preconditions
- User B is logged in
- Push notifications are enabled for the app
- App is in background on User B device
- User A can send messages to User B

## Steps to Reproduce
1. Put the app in background on User B device
2. From User A, send one message to User B
3. Observe push notifications on User B device

## Actual Result
Two notifications appear for one message.

## Expected Result
Only one notification should be received per single message event.

## Environment
- Device: Android Emulator (Pixel 6)
- OS: Android 13
- App Build: Staging
- Network: Wi-Fi
