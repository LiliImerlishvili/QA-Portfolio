**Bug ID:** BUG-001  
**Priority:** High  
**Severity:** Major  
**Status:** Open  

## Description
The chat module allows sending a message when the input contains only whitespace characters. This creates empty-looking messages and degrades user experience.

## Preconditions
- User A is logged in
- User A has an existing chat with User B
- Chat screen is opened

## Steps to Reproduce
1. Tap on the message input field
2. Enter several spaces (example: "     ")
3. Tap the "Send" button

## Actual Result
A blank/empty-looking message is sent and appears in the chat thread.

## Expected Result
The app should prevent sending whitespace-only messages and show a validation hint (example: "Message cannot be empty").

## Environment
- Device: Android Emulator (Pixel 6)
- OS: Android 13
- App Build: Staging
- Network: Wi-Fi
