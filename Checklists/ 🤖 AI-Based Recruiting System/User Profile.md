## 2. Profile Page Accessibility Checklist

**ID:** AI-REC-CL-002  
**Module:** User Profile

- [ ] Logged-in user can access profile page  
- [ ] Profile loads without “User not found” error  
- [ ] User information (name, role, email) appears correctly  
- [ ] Navigation from Settings → Account → Profile works  
- [ ] Page loads within expected response time  
- [ ] No broken UI elements on the profile page
