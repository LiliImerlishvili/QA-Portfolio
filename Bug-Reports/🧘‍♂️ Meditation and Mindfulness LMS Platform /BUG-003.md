# 🐞 BUG-003 – Stripe Checkout Fails Without Error Message

**Priority:** High  
**Severity:** Major  

## Description
Stripe payment fails silently without displaying any error feedback to the user.

## Preconditions
- User is logged in
- Stripe payment method selected
- Invalid or declined card used

## Steps to Reproduce
1. Navigate to checkout
2. Select Stripe as payment method
3. Enter invalid card details
4. Submit payment

## Actual Result
Checkout reloads without confirmation or error message.

## Expected Result
Clear error message should inform the user about payment failure.
