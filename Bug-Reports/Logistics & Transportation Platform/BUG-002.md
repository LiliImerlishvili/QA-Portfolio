# 🐞 BUG-002 – Incorrect route data displayed after language switch

**Priority:** Medium  
**Severity:** Major  

## Description
Changing the application language causes incorrect route data to be displayed.

## Preconditions
- User is on route details page
- Multiple languages are enabled

## Steps to Reproduce
1. Select a route
2. Change the language using the language switcher

## Actual Result
Displayed route details do not match the selected route.

## Expected Result
Route data should remain consistent after language change.
