# 🐞 BUG-006 – Map zoom level resets after route selection

**Priority:** Low  
**Severity:** Minor  

## Description
The map zoom level resets when a route is selected.

## Preconditions
- Map is loaded

## Steps to Reproduce
1. Zoom in on the map
2. Select a route

## Actual Result
Map zoom resets to default.

## Expected Result
Zoom level should remain unchanged.
