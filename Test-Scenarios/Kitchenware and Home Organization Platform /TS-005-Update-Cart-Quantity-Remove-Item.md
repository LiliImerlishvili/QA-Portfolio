# TS-005 - Update Cart Quantity and Remove Items

## Goal
Verify that cart updates (quantity changes, remove item) work correctly and totals update accordingly.

## Preconditions
- At least one product is already added to cart.
- Cart page or cart drawer is accessible.

## Flow (High-level)
1. User opens cart page.
2. User increases item quantity.
3. User decreases item quantity.
4. User removes an item from cart.
5. User verifies totals and cart state update correctly after each change.
