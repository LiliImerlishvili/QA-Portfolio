**Bug ID:** BUG-005  
**Priority:** High  
**Severity:** Critical  
**Status:** Open  

## Description
When a user uploads an image in chat and switches the app to background during upload, the upload gets stuck on "Uploading" and never completes, even after returning to the app.

## Preconditions
- User is logged in
- Chat screen is opened
- Photo permission is granted
- Network is stable

## Steps to Reproduce
1. Start uploading an image in chat
2. Immediately switch app to background (Home button)
3. Wait 10-15 seconds
4. Return to the app
5. Observe upload state

## Actual Result
Upload remains stuck on "Uploading" indefinitely and message is not sent.

## Expected Result
Upload should either complete in background, resume when app returns, or fail gracefully with a clear error and retry option.

## Environment
- Device: Android Emulator (Pixel 6)
- OS: Android 13
- App Build: Staging
- Network: Wi-Fi
