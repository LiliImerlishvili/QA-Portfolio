# 🐞 BUG-001 – Route map does not load on initial page visit

**Priority:** High  
**Severity:** Major  

## Description
The route map does not load when the user opens the route selection page for the first time.

## Preconditions
- User is logged in
- Route selection page is accessible

## Steps to Reproduce
1. Open the route selection page
2. Observe the map container

## Actual Result
The map area remains blank and route data is not displayed.

## Expected Result
The map should load automatically and display available routes.
