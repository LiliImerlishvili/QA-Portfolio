# 🚚 Logistics & Transportation Platform (Confidential Client)

This project represents manual QA work performed on a **Logistics & Transportation Platform**
focused on route management, booking workflows, delivery tracking, and multi-language support.

The platform supports operational logistics flows across different regions and languages,
requiring high reliability, accurate data handling, and stable integrations with third-party services.

---

## 👩‍💻 Role & Responsibilities

As a **Manual QA Engineer**, I was responsible for validating core logistics workflows,
ensuring localization accuracy, and verifying system stability across browsers and environments.

Key responsibilities included:
- Functional testing of route selection, booking, and delivery workflows
- Localization testing across multiple languages
- Validation of map integrations and language switcher behavior
- Regression testing after fixes and updates
- End-to-end scenario validation for logistics operations

---

## 🧪 Testing Scope

Testing activities covered the following areas:

- Route selection and navigation flows  
- Booking and delivery lifecycle validation  
- Multi-language UI and content behavior  
- Map integrations and error handling  
- Cross-browser and cross-device consistency  
- Regression and retesting cycles

## 🧰 Tools Used
